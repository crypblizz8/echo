# Prerequisites

## Supabase

- Create new project
- Get SUPABASE_URL and SUPABASE_ANON_KEY
- On Dashboard --> Project Settings --> Authentication

## ENVS

- Reference env.example
- Add into env

## Installation

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
